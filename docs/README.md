## 🎯 Démarrage Rapide

### Installation Basique

```bash
# 1. Cloner
git clone https://github.com/votre-repo/mayegue-mobile.git

# 2. Installer
cd mayegue-mobile
flutter pub get

# 3. Lancer
flutter run
```

